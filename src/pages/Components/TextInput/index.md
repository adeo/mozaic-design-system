## Do's and don'ts

<hintitem>
    Input text must have an explicit label.
</hintitem>

<hintitem>
    Use Input text for monoline text. For multiple lines, use a textarea instead.
</hintitem>

<hintitem>
    Always use a label associated with an input, even using placeholders
</hintitem>
