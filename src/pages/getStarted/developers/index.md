# Installation :

### Requirements :

Installing Garden require `node v10 +`

### Initialise a new project (if it's not already done):

```BASH
# got to your project
cd my/project/path
# if it is not already done, initialize npm
npm init
```

and follow the npm init instructions

### install garden :

```bash
npm install --save garden-css
```
