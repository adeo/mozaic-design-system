## Accessibility and semantic

<HintItem>
  The attribute <strong>aria-expanded="false"</strong> must{' '}
  <strong>always be present</strong> on the{' '}
  <strong>.mc-accordion__label</strong> element when the accordion is closed.
</HintItem>
<HintItem>
  Don't forget to change the value of the <strong>aria-expanded</strong> attribute
  from <strong>"false"</strong> to <strong>"true"</strong> on the{' '}
  <strong>.mc-accordion__label</strong> element when it's open
</HintItem>
