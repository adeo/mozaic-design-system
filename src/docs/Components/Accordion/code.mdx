## Import

Import *the settings* and *the accordion* scss files. 
```scss
@import 'settings-tools/_all-settings';

@import 'components/_c.accordion';
```

## Basic usage
To create a basic accordion, apply the following classes on this HTML elements:

- `mc-accordion` on a div container
- `mc-accordion__trigger` on a input[checkbox] tag
- `mc-accordion__label` on a label tag
- `mc-accordion__content` on a div tag

```html
<div class="mc-accordion">
  <button id="accordion" class="mc-accordion__trigger" aria-expanded="false" aria-controls="content">
    <span class="mc-accordion__label">Accordion title</span>
  </button>
  <div id="content" class="mc-accordion__content">
    <p>
      Fusce iaculis dolor nulla, a maximus ipsum sollicitudin et. Ut
      condimentum at orci aliquam feugiat. Curabitur sagittis placerat leo sit
      amet pharetra.
    </p>
  </div>
</div>
```

## Variations

### Sizes

Mozaic offers you two sizes of accordions that you can use, by applying the following modifiers on the `mc-accordion` element :

- Size **S**: use the `mc-accordion--s` modifier
- Size **M**: default size - no modifier needeed

<Preview path="sizes" />

### No padding

You have the possibility to have an accordion with no padding.

For this you have to add to the label the `mc-accordion__trigger--no-padding` modifier.

<Preview path="no-padding" />

### With icon

You can add an icon into any accordions you want.

For this you have to add to the label the `mc-accordion__trigger--icon` modifier.

Then just add the CSS class `mc-accordion__icon` to your icon.

```html
<div class="mc-accordion">
    <div class="mc-accordion__header">
      <button
        class="mc-accordion__trigger mc-accordion__trigger--icon "
        aria-expanded="false"
        aria-controls="content"
        id="accordion"
      >
        <span class="mc-accordion__label">
          <svg class="mc-accordion__icon">
            <use href="#Store_StoreLM_32px"></use>
          </svg>
          Accordion title</span
        >
      </button>
    </div>
    <div id="content" class="mc-accordion__content">
      <p>
        Fusce iaculis dolor nulla, a maximus ipsum sollicitudin et. Ut
        condimentum at orci aliquam feugiat. Curabitur sagittis placerat leo sit
        amet pharetra.
      </p>
    </div>
  </div>
</div>
```

### With Toogle

You can add a Mozaic toggle into any accordions you want.

For this you have to add to the label the `mc-accordion__header` modifier.

Then add the CSS class `mc-accordion__selectable` to your toogle.

```html
<div class="mc-accordion">
    <div class="mc-accordion__header">
      <button class="mc-accordion__trigger" aria-expanded="false" aria-controls="content" id="accordion">
        <div class="mc-accordion__label">
          <div class="mc-toggle mc-accordion__selectable">
            <input class="mc-toggle__input" id="togg" type="checkbox" />
            <label class="mc-toggle__label" for="togg">
          </div>
          <span>Accordion with toggle</span>
        </div>
      </button>  
    </div>
    <div id="content" class="mc-accordion__content">
      <p>
        Fusce iaculis dolor nulla, a maximus ipsum sollicitudin et. Ut
        condimentum at orci aliquam feugiat. Curabitur sagittis placerat leo sit
        amet pharetra.
      </p>
    </div>
  </div>
```

### With Checkbox

You can add a Mozaic checkbox into any accordions you want.

For this you have to add to the label the `mc-accordion__header` modifier.

Then just add the CSS class `mc-accordion__selectable` to your checkbox.

Finally add the class `mc-accordion__label--checkable` to the span element.

```html
  <div class="mc-accordion">
    <div class="mc-accordion__header">
      <button class="mc-accordion__trigger" aria-controls="content" id="accordion">
        <div class="mc-accordion__label">
          <div class="mc-checkbox mc-accordion__selectable">
            <input
              id="checkbox-01"
              type="checkbox"
              class="mc-checkbox__input"
              name="example"
              aria-label="checkboxlabel"
            />
          </div>
          <span>
            Accordion with checkbox
          </span>
        </div>
      </button>
    </div>
    <div id="content" class="mc-accordion__content">
      <p> 
        Fusce iaculis dolor nulla, a maximus ipsum sollicitudin et. Ut
        condimentum at orci aliquam feugiat. Curabitur sagittis placerat leo sit
        amet pharetra.
      </p>
    </div>
  </div>
```
