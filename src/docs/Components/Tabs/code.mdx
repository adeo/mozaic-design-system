## Basic implementation

### Import

Import **the settings** and **the tabs** `scss` files.

```scss
@import 'settings-tools/_all-settings';

@include import-font-families();

@import 'components/_c.tabs';
```

### Basic usage

To create a basic tabs, wrap a link `a` inside a `ul` and apply the following classes:

- `mc-tabs` on a div container
- `mc-tabs__nav` on the ul tag
- `mc-tabs__item` on the li tag
- `mc-toggle__link` on the a tag

```html
	<div class="mc-tabs">
		<ul class="mc-tabs__nav" role="tabs-navigation">
			<li
				class="mc-tabs__nav-item mc-tabs__nav-item--selected"
				data-target=".tab-1-content" role="tab"  aria-selected="true">
				<a tabindex="0" class="mc-tabs__nav-link" href="javascript:void(0);" role="tab">Tab Selected</a>
			</li>
			<li
				class="mc-tabs__nav-item "
				data-target=".tab-2-content" role="tab">
				<a tabindex="0" class="mc-tabs__nav-link" href="javascript:void(0);" role="tab">Tab Default 1</a>
			</li>
			<li
				class="mc-tabs__nav-item "
				data-target=".tab-3-content" role="tab">
				<a tabindex="0" class="mc-tabs__nav-link" href="javascript:void(0);" role="tab">Tab default 2</a>
			</li>
			<li
				class="mc-tabs__nav-item  mc-tabs__nav-item--disabled "
				data-target=".tab-4-content" role="tab"  
				aria-disabled="true">
				<a tabindex="0" class="mc-tabs__nav-link" href="javascript:void(0);" role="tab">Tab Disabled</a>
			</li>
		</ul>
	</div>
```

#### Accessibility and semantic

<HintItem>
Content authors need to ensure the content that is added to the tab panel is accessible. For example, if you add an image to the panel you need to include alternative text to pass accessibility testing.
</HintItem>

<HintItem>
Don't forget to add the attribute role="tablist" on the mc-tabs__nav element<br/>
</HintItem>

<HintItem>
Don't forget to add the attribute role="presentation" on the mc-tabs__item element<br/>
</HintItem>

<HintItem>
Don't forget to add the attribute role="tab" on the mc-tabs__link element<br/>
</HintItem>
