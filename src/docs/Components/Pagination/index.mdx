> The pagination enable users to navigate from a page to another one, especially when you have to split a large content in several parts. It's mainly used on pages like the search engine result page or the list pages.

## Variations

### Horizontal navigation

There is two mechanics in this case. You can use the arrows to go on previous or next pages, or you can used the select to go on the wanted page.

--- Preview ---

**Horizontal pagination give you theses following informations:**

- A visual information to indicate the page where you are.
- A button to navigate to the previous page
- A button to navigate to the next page.
