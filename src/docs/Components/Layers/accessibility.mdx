## Accessibility

<HintItem>
  The layer container (<strong>mc-layer</strong>) must always have a{' '}
  <strong>aria-labelledby</strong> attribute whose value corresponds to the
  value of the <strong>id</strong> attribute of the layer title.
</HintItem>
<HintItem>
  The layer title must always have an <strong>id</strong> attribute.
</HintItem>
<HintItem>
  The layer container (<strong>mc-layer</strong>) must always have a{' '}
  <strong>aria-modal</strong> attribute whose value is <strong>true</strong>.
</HintItem>
<HintItem>
  When layer is hidden, the <strong>mc-layer</strong> container must have a{' '}
  <strong>aria-hidden="true"</strong> attribute.
</HintItem>
<HintItem>
  When layer is visible, the <strong>mc-layer</strong> container must have a{' '}
  <strong>aria-hidden="false"</strong> attribute.
</HintItem>
