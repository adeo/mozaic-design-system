## Accessibility

<HintItem>
  Don't forget to set <code>role="alert"</code> in the notification when It's
  used to alert about a result of user action
</HintItem>

```html
<div class="mc-notification" role="alert">
  <div class="mc-notification__content">
    ...
  </div>
</div>
```

<HintItem>
  Always use a title attribute on the close button <code>title="Close"</code>
</HintItem>

```html
<div class="example">
  <div class="mc-notification mc-notification-closable">
    <div class="mc-notification__content">
      ...
    </div>
    <button
      type="button"
      class="mc-notification-closable__close"
      title="Close"
    ></button>
  </div>
</div>
```
