## Accessibility attributes

The ProgressBar component has been designed to best meet current accessibility criteria.

Therefore, in order to guarantee the accessibility of the component, please make sure to respect the following rules when integrating the component into your pages:

<HintItem>
  The attribute <strong>role="progressbar"</strong> must{' '}
  <strong>always be present</strong> on the{' '}
  <strong>.mc-progressbar__indicator</strong> element.
</HintItem>
<HintItem>
  The attribute <strong>aria-valuemin</strong> must{' '}
  <strong>always be present</strong> on the{' '}
  <strong>.mc-progressbar__indicator</strong> element. <br />
  This attribute defines the minimum value the progression.
</HintItem>
<HintItem>
  The attribute <strong>aria-valuemax</strong> must{' '}
  <strong>always be present</strong> on the{' '}
  <strong>div.mc-progressbar__indicator</strong>. <br />
  This attribute defines the maximum allowed value of the progress bar.
</HintItem>
<HintItem>
  If possible, the attribute <strong>aria-valuenow</strong> should be present on
  the <strong>div.mc-progressbar__indicator</strong>. <br />
  This attribute indicates in real time the current value of the progression. <br />
  If the progression value is undetermined, the developer can omit this
  attribute.
</HintItem>
