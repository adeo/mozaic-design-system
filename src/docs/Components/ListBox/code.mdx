## Import

Import the settings and the listbox `scss` files.

```scss
@import 'settings-tools/all-settings';
@import '_c.listbox'; // mandatory

@import '_c.checkbox'; // optionnal
```

## Usage

The creation of an listbox component in your code is done using the following HTML structure:

```HTML
  <ul
    tabindex="-1"
    role="listbox"
    aria-labelledby="phone_number_list"
    class="mc-list-box"
  >
    <li class="mc-list-box__item" role="option">
      <span class="mc-list-box__text">Option label</span>
    </li>
  </ul>
	
```
