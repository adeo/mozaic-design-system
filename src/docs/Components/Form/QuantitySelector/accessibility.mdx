## Sizes

Please know that the `small` version is not really supposed to be used on touch devices to avoid misclicking. We recommend some blank space before and after the quantity selector to have the best user experience possible.


## Accessibility and semantic
<HintItem>
The buttons element (mc-button) must always have a aria-controls attribute whose value corresponds to the value of the `id` attribute of the input field element.
</HintItem>
<HintItem>
The input field element must always have an attribute id
</HintItem>
<HintItem>
Use <strong>aria-valuemin</strong> attribute to set the minimum value allowed for the quantity selector.
</HintItem>
<HintItem>
Use <strong>aria-valuemax</strong> attribute to set the maximum value allowed for the quantity selector.
</HintItem>
