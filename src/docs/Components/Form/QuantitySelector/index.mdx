The quantity selector is form element used to enter or select a number. You can either type the number in the field or use the plus and minus icon to select the wanted number.

## Variations

We provide two different sizes to adapt to your design needs but keep in mind that the `medium` case must be used on smaller breakpoints. More on [accessbility](Components/Form/QuantitySelector/accessibility/) page.

<Preview path="basic" />

## Usage

This type of input is best used when the user need to choose the quantity of a selected item, like a product before adding to cart for example.

<Highlight theme="tips">

  Dont forget that the user can use its keyboard or the incremental buttons to select the quantity.

</Highlight>
