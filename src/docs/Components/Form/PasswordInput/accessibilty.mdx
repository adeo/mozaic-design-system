## Accessibility

In order to guarantee the component accessibility, please make sure to respect the following rules when integrating the component into your pages:

<HintItem>
  Always use a <strong>role="switch"</strong> attribute on the{' '}
  <strong>.mc-password-input__button</strong> element.
</HintItem>

<HintItem>
  Always use an <strong>aria-pressed</strong> attribute on the{' '}
  <strong>.mc-password-input__button</strong> element, and dynamically change
  its value from <strong>false</strong> to <strong>true</strong> and vice versa
  depending on the state of the button.
</HintItem>

```html
<button
  type="button"
  class="mc-password-input__button"
  role="switch"
  aria-pressed="false"
>
  Show
</button>
```
