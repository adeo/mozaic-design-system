## Accessibility and semantic
<HintItem>
The button element (mc-phone-number__button) must always have an <strong>id</strong>, a <strong>aria-haspopup</strong>, <strong>aria-labelledby</strong> and the <strong>type="button"</strong> attribute.
</HintItem>
<HintItem>
The list element (mc-phone-number__list) must always have a <strong>role="list"</strong> or <strong>role="listbox"</strong> attribute.
</HintItem>
<HintItem>
Each list element must have the <strong>role="option"</strong>
</HintItem>
<HintItem>
Add the <strong>aria-selected="true"</strong> attribute to the selected list element.
</HintItem>
