## Import

Import *the settings*, *the grid* and *the hero* scss files. 
```scss
@import 'settings-tools/_all-settings';

@include import-font-families();

@import 'typography/_t.all-base-typography';
@import 'layouts/_l.flexy';
@import 'layouts/_l.container';
@import 'components/_c.hero';
```

## Basic usage
A Hero is composed of several zones:

- *A body* with the class: `mc-hero__body`
- *A header* with the class: `mc-hero__header`
- *A content* with the class: `mc-hero__content` (optional)
- *A background image* with the class: `mc-hero__background`

These areas are encompassed by a global wrapper element with class mc-hero

```html

<div class="mc-hero ml-flexy ml-flexy--items-center">
  <div class="mc-hero__body mc-hero__body--with-content">
      <div class="mc-hero__header">
        <h1 class="mt-heading
        mt-heading--lightest
        mt-heading--s
        mt-heading--underline">Hero title</h1>
    </div>
    <div class="mc-hero__content mt-body-m">
      <p> Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.</p>
    </div>
  </div>

  <div class="mc-hero__background">
    <picture>
      <img class="mc-hero__img" src="https://m1.lmcdn.fr/media/15/5e611393475ea771bbbad626/2872342965/greenkub-maison-sans-permis5e611393063b330009a4a405.jpg?width=800&crop=16:9,smart" alt="Greenkub house">
    </picture>
  </div>
</div>

```

### Details of area
In more detail, it should be noted that each of the fields mentioned above may contain other elements. Thus:

#### Main
In order to have a clean layout you will have to add the following classes to the container *`mc-hero`*:
- `ml-flexy ml-flexy--items-center`

#### The background
The background block contains:
- An image element (`mc-hero__img`) inside a picture element
- Before the img element you can add all the source for different screen size

Here a basic implementation of a Picture Element

``` html 
<picture>
	<source srcset="here-your-800-image.jpg" media="(min-width: 800px)">
	<source srcset="here-your-1200-image.jpg" media="(min-width: 1200px)">
	<img src="here-your-default-image.jpg" alt="" />
</picture>
```

Learn more about this on [MDN Documentation](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/picture)

#### Body
The body contains: 
 - the header (`mc-hero__header`)
 - the content (`mc-hero__content`)

#### The header
The header contains:
- the hero title (.mc-hero__header) 
- the title element (h1) must have these classes: `mt-heading mt-heading--lightest mt-heading--s mt-heading--underline`

#### The content
- Apply `mt-body-m` on `mc-hero__content`

### Centered hero
To center title and content you will have to apply these following classes: 
- Add `mc-hero--center` modifier to `mc-hero` element
- Add `mc-hero__body--center` modifier to `mc-hero__body` element
- Add `mt-heading--center` modifier to `mt-heading` element
- Use `mc-hero__body--center-with-content` modifier instead `mc-hero__body--center` for title with content variation.

<Preview path="centered" nude/>

### Box hero

- Add `mc-hero--with-box` modifier to `mc-hero` element	
- Add your box content inside `mc-hero__box`

#### Box hero modifier
There is 2 modifier available for the box hero. 

- `mc-hero--with-box`: the box is placed aside from `screen.l`
