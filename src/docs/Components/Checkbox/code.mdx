## Import

Import the settings and the checkbox `scss` files.

```scss
@import 'settings-tools/_all-settings';

@include import-font-families();

@import 'components/_c.checkbox';
```

## Basic usage
To create a classic checkbox you have 3 main classes to apply:

- `mc-checkbox` on div
- `mc-checkbox__input` on input
- `mc-checkbox__label` on label

```html
<input id="checkbox-01" type="checkbox" class="mc-checkbox" name="example" />
<label for="checkbox-01" class="mc-checkbox__label"> Label </label>
```

### Standard input states :

- `:hover`
- `:focus`
- `:checked`
- `:disabled`
- `.is-invalid`
