> A body style in typography is the main text part of a pattern or a page. It must be easy to identity and provide basic typographic styles for readable multilines texts or paragraphs.

## Recommendation

### Body Medium

The default body style is **body medium** with the following specifications:

| Font-size           | Font-weight           | Line-height           |
| ------------------- |---------------------- | --------------------- |
| 16px                | Regular               | 1.35                  |
