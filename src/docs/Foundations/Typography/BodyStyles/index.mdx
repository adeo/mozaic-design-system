> A body is a paragraph or a multiline text. It provides basic typographic properties like font-size, line-height and font-weight.

## Recommended size

### Body Medium

The default body style is **body medium** (`mt-body-m`) with the following specifications:

| Font-size           | Font-weight           | Line-height           |
| ------------------- |---------------------- | --------------------- |
| 16px                | Regular               | `$line-height l`      |

## Specific needs

Mozaic's typographic system still allows you to answer specific needs in some cases like smaller or larger font-sizes and also thicker font-weight like **semibold**.

### Body Small

For accessibility and readability, this is the smallest value you can use for multiline texts.

| Font-size           | Font-weight           | Line-height           |
| ------------------- |---------------------- | --------------------- |
| 14px                | Regular               | `$line-height l`      |

<Preview path="typographyBodySmall" nude />

### Body Large

This value must be the biggest you use to have readable paragraphs on every device.

| Font-size           | Font-weight           | Line-height           |
| ------------------- |---------------------- | --------------------- |
| 18px                | Regular               | `$line-height l`      |
