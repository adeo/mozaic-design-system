> A body style in typography is the main text part of a pattern or a page. It must be identity and provide basic typographic styles for readable multilines texts or paragraphs.

## Recommendation

For readability and accessibility purposes we recommend you to use our regular value for body styles called **body medium**, corresponding to the following values:

- Font-size: 16px
- Font-weight: Regular
- Line-height: 1.35 (= M)

## Specific needs

Each body pattern comes with a modifier to set the font-weight to **'semi bold'** and a modifier to use an alternative **line-height**. You will also find smaller and bigger values to answer specific needs.
