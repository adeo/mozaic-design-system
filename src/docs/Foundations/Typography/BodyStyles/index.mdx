> A body style in typography is the main text part of a pattern or a page. It must be easy to identity and provide basic typographic styles for readable multilines texts or paragraphs.

## Recommendation

### Body Medium

The default body style is **body medium** (`mt-body-m`) with the following specifications:

| Font-size           | Font-weight           | Line-height           |
| ------------------- |---------------------- | --------------------- |
| 16px                | Regular               | `$line-height m`      |

## Specific needs

Mozaic's typographic system still allows you to answer specific needs in some cases like smaller or larger font sizes and also thicker font weight like **semibold**.

### Body Small

For accessibility and readability, this is the smallest value you can use for this kind of text.

| Font-size           | Font-weight           | Line-height           |
| ------------------- |---------------------- | --------------------- |
| 14px                | Regular               | `$line-height m`      |

<Preview path="typographyBodySmall" nude />

### Body Large

This value must be the biggest you use to be readable on every device.

| Font-size           | Font-weight           | Line-height           |
| ------------------- |---------------------- | --------------------- |
| 18px                | Regular               | `$line-height m`      |
