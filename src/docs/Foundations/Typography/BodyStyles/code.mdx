## Import

```scss
@import 'settings-tools/_all-settings';

@include import-font-families();

@import 'typography/_t.body';
```

## Basic usage

To use the body style pattern you only have to use the main css class `mt-body`.

You can apply this class directly to the desired HTML element or you can apply it to an HTML element used as a wrapper.

```html
<p class="mt-body">Lorem ipsum...</p>

<div class="mt-body">
  <p>Lorem ipsum...</p>
  <p>Lorem ipsum...</p>
  <p>Lorem ipsum...</p>
</div>
```
