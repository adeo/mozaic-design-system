> The container defines the main content width as well as the minimum negative space between the horizontal edges of the screen and the content.

## Default container

The default container follows these rules :

| Viewport       | Container width          | External margins (on each side) |
| -------------- | ------------------------ | ------------------------------- |
| `s` > 0px      | Fluid                    | `1mu` (16px)                    |
| `m` > 680px    | Fluid                    | `2mu` (32px)                    |
| `l` > 1024px   | `58mu` (928px) / Fluid   | `3mu` (48px)                    |
| `xl` > 1280px  | `76mu` (1216px) / Fluid  | `2mu` (32px)                    |
| `xxl` > 1920px | `115mu` (1840px) / Fluid | `2.5mu` (40px)                  |

<br />
<small>Reminder : mu = MagicUnit. Ex: 3.5mu = MagicUnit \* 3.5</small>

## The fluid container

The fluid container uses the values of the same padding but is fluid at any viewport width.
