## Import

Import **the container** scss files.

```scss
@import 'layouts/l.container';
```

## Usage

### Default container

To use a **default container** in your code, simply add the `ml-container` class to the HTML element of your choice, as follows:

```html
<main class="ml-container">...</main>
```

### Fluid container

Use the `ml-container--fluid` modifiers if you want to use the default paddings only but still want the container to be fluid at any viewport width.

```html
<main class="ml-container ml-container--fluid">...</main>
```
