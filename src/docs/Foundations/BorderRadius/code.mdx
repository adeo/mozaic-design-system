## Import

To get the `set-border-radius()` mixin, you need to import `_all-settings.scss`.

```css
// mandatory
@import 'settings-tools/_all-settings';
```

## Basic usage

```css
.example {
    @include set-border-radius('s');
    @include set-border-radius('m');
    @include set-border-radius('l');
}
```
