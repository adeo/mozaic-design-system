A new mecanic has been set up to cover accessibility need in a smart way.
Please, take notes that the shared contrast ratio is based on white background.

Please, apply the following rules :

Nous avons mis en place une mécanique qui s'adaptera de la même manière sur toutes les cpuleurs.
Veuillez prendre notes que le use case principal est sur fond blanc. A partir de ce use case, la couleurs de ref est le -600.

Voici les règles suivantes

| Element                                                       | Foreground | Background | Ratio |
| ------------------------------------------------------------- | ---------- | ---------- | ----- |
| Text                                                          | `-600`     | `Grey-000` | 4.5:1 |
| Icon or graphic elements with informative or functional usage | `-500`     | `Grey-000` | 3:1   |
