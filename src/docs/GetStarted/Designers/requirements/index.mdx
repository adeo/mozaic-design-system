## Adeo Github access

Adeo Github access is required for download the Mozaic UI Kit.

[Join Adeo Github](https://adeo.github.io/join/) and follow the steps.<br />
_You will have to create an account if you don't already have one._

## Sketch App - Version 57.1

All our patterns are currently designed on **Sketch**.

[Download Sketch 57.1.](https://www.sketchapp.com/updates/#version-57.1)

### Sketch plugins

By using Mozaic **you don't depend on plugins**. We don't recommend the use of plugins like Anima.
