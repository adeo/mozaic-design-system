## Prerequisites

1- Get the BrowserStack API KEYs

To obtain your username and access keys sign up for a Free Trial: https://www.browserstack.com/users/sign_up

2- Set the `BROWSERSTACK_USERNAME` & `BROWSERSTACK_KEY` environment variable

- Add your API keys in your `.env` file 
    `BROWSERSTACK_USERNAME='YOUR_API_USERNAME'`
    `BROWSERSTACK_KEY='YOUR_API_KEY'`
or
- Set the API key as environment variable 
    - Mac: `export BROWSERSTACK_USERNAME='YOUR_API_KEY' BROWSERSTACK_KEY='YOUR_API_KEY'`
    - Windows: `set BROWSERSTACK_USERNAME='YOUR_API_KEY' BROWSERSTACK_KEY='YOUR_API_KEY'`

## Run tests (locally)

- `yarn test`
or
- `npx wdio`

> **CAUTION:** The test will create a screenshots for capabilities available and save it in `test/screenshots` folder

> **CAUTION:** To run tests locally make sure you have a mozaic app running at localhost:8000.
